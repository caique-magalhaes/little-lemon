# Little Lemon Restaurant Landing Page

A clean, responsive restaurant landing page built to demonstrate modern web layout architecture and fluid user interface interactions using native web technologies.

## 🚀 Live Demo
🌐 **View the live website here:** [caique-magalhaes.github.io/little-lemon/](https://github.io)

---

## 🛠️ Tech Stack & Key Implementations

*   **Structure:** HTML5 (Semantic elements for clean document architecture)
*   **Layout Engine:** CSS3 Flexbox (Used extensively to manage fluid navigation bars, flexible food menu grids, and multi-device section alignments)
*   **Interactions & Animations:** CSS3 Transitions & Transforms
    *   Features smooth hover states across interactive cards and buttons.
    *   Implements `transform: scale(1.02);` combined with transition timing functions to create a responsive, tactile lifting effect on user hover.

## 💻 Local Setup & Development

To view this project locally on your machine:

1. Clone this repository:
   ```bash
   git clone https://github.com
   cd little-lemon
   ```

2. Open the entry point file in any browser:
   ```bash
   # On Windows:
   start index.html
   
   # On macOS/Linux:
   open index.html
   ```
